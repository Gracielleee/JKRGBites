Application Description
JRKG Bites is a mobile application that will help a user find a perfect match for their dining desires, whether it be based on craving, mood, or preference. The app solves the very common problem of decision fatigue when it comes to where and what to eat through an interactive, personalized experience that caters to a single user's preferences.

Problem Statement
The application aims to ease the decision-making process for users who struggle with choosing dining options. By offering personalized recommendations based on everyday experiences and user interactions, JRKG Bites streamlines the process of finding satisfying meal options.

Home Page
The Home page features a card-based interface displaying restaurants from the JSON database, filtered by user location.

Key Features:
Automatic filtering based on user's proximity/radius
Card interaction: Single tap flips card to reveal details including restaurant ID, name, available locations, category, tags, and cuisine
Zoom-in animation for enhanced user experience

Location Services
Google Maps API Integration: Determines user's exact or nearest estimated location
Proximity Filter: Displays restaurants within user-defined radius
Navigation Support: Provides walking directions for restaurants within 2-kilometer limit

Gesture-Based Selection System
The application employs an intuitive gesture-based interface similar to popular swipe-based apps, allowing users to make quick decisions about dining options.

Gesture Actions:
Swipe Up: Save restaurant to Favorites (personalized per user account)
Swipe Down: Move to "Never Again" list (soft deletion; removed from future selections unless re-added via Search)
Swipe Left: Temporarily discard from current selection (reappears on app restart)
Swipe Right: Select restaurant and view complete information including ID, name, location details (mall/market/city), cuisine type, floor level, and coordinates with optional Google Maps navigation
Single Tap: Flip card to view details; tap again to return to logo view
Shake Device: Activate roulette function
Gamification - Roulette Feature
Activation: Shake device on specific pages (Home or Favorites)
Functionality: Randomly selects a restaurant from the current page's list
Dynamic Lists: Automatically includes restaurants based on current page context

Search Function
Comprehensive search across all restaurants in database
Includes restaurants in "Never Again" list
Re-listing capability: Users can restore "Never Again" restaurants by tapping the restaurant name and selecting "Add Back to List"
Accessible via bottom navigation bar
